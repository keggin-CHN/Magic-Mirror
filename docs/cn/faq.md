# 常见问题

## 下载卡住怎么办？

可以前往仓库 Release 页面手动下载服务端压缩包：

https://github.com/keggin-CHN/Magic-Mirror/releases

模型下载地址保持不变。

## 没检测到人脸怎么办？

请更换更清晰、光照更好、无遮挡的正脸照片。

## 视频处理很慢怎么办？

尝试启用 GPU 加速。Windows 推荐 DirectML，NVIDIA 显卡可尝试 CUDA。

## 输出视频没有音频怎么办？

请确认系统中可以使用 ffmpeg，或尝试更换源视频编码。

## 默认登录密码是什么？

默认密码为 `123456`，登录后可以修改。
