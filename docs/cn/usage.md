# 使用说明

## 基本换脸流程

1. 启动前端和 Python 服务端。
2. 上传一张清晰的正脸照片作为人脸来源。
3. 上传目标图片或视频。
4. 选择单人换脸或多人换脸模式。
5. 点击开始换脸并等待处理完成。
6. 点击下载结果保存输出文件。

## 多人换脸

在多人模式中，可以添加多张人脸素材，并为不同选区指定不同的人脸来源。

## 视频换脸

视频处理耗时较长，建议启用 GPU 加速。Windows 推荐 DirectML，NVIDIA 显卡可尝试 CUDA。

## 常见注意事项

- 建议使用清晰、无遮挡、正脸图片。
- 不支持 HEIC/HEIF，建议转换为 JPG 或 PNG。
- 视频建议使用 MP4/H.264 编码。
