# FAQ

## What if downloading gets stuck?

Download the server package manually from the repository releases page:

https://github.com/keggin-CHN/Magic-Mirror/releases

Model download URLs remain unchanged.

## What if no face is detected?

Use a clearer, well-lit, unobstructed front-facing photo.

## Why is video processing slow?

Enable GPU acceleration if available. DirectML is recommended on Windows, and CUDA may work on supported NVIDIA GPUs.

## Why does the output video have no audio?

Make sure ffmpeg is available, or try another source video encoding.

## What is the default login password?

The default password is `123456`. You can change it after logging in.
